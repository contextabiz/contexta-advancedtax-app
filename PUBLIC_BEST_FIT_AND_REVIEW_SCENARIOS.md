# Public Best-Fit And Review Scenarios

This document is a client-friendly explanation of when the app is strongest and when a human review is still recommended.

## Best-Fit Scenarios

The app is currently most accurate and most efficient for returns such as:

- employment-focused returns using `T4`
- personal returns using common slips such as `T3`, `T4A`, `T4PS`, `T5`, and `T2202`
- returns with common deductions such as RRSP, FHSA, RPP, union dues, moving expenses, and carrying charges
- returns with common credits such as tuition, medical expenses, donations, and dividend tax credits
- returns with moderate investment income, including interest, dividends, and taxable capital gains
- returns with foreign income and foreign tax paid that fit the app's `T2209 / T2036` workflow
- returns with moderate household claim review, including spouse, eligible dependant, caregiver, disability transfer, and dependant medical logic
- Ontario-focused returns, and also B.C. or Alberta returns, where the worksheet depth is strongest

## Typical Strong Use Cases

These are especially good fit scenarios:

- `T4-only` returns
- `T4 + T2202` returns
- `T4 + T3 / T4PS / T5` investment-slip returns
- `T4 + RRSP / FHSA / donations / medical` returns
- personal returns that need a guided estimate, review summary, or preparer-style workpaper

## Situations That Still Need Human Review

A human review is still recommended when a return includes:

- unusual household or dependant situations
- multiple dependants with overlapping claim choices
- complex foreign-tax situations
- highly unusual provincial credits or province-specific schedules
- special exceptions, edge-case elections, or uncommon CRA worksheet outcomes
- situations where supporting slips and worksheets do not reconcile cleanly
- cases where the app shows `Review` or `Missing` items in the filing-readiness checklist

## Situations Where This App Is Not The Best Fit

This app is not currently the best fit for:

- `T2125` self-employment or sole proprietorship returns
- business-use-of-home or business vehicle allocation
- inventory-based business returns
- full business tax workflows
- CRA-certified filing workflows

## Safe Client-Facing Positioning

If you need a short explanation for clients, this is a safe version:

`This tool is strongest for personal T1 tax estimates built from common slips, common deductions and credits, and standard personal-return schedules. More unusual or highly complex cases should still be reviewed manually.`

## Shorter Version

`Best for common personal T1 returns. More complex or unusual returns should still be manually reviewed.`
